API REST sobre rifles de francotirador

DESCRIPCION
Esta es una API basica dedicada a un tema de mi interes, en este caso sobre rifles de francotirador.

TECNOLOGIAS
NODE.JS (para levantar el servidor)
EXPRESS (para la creacion de los endpoints y las rutas)

FUNCIONAMIENTO
El usuario puede crear, leer, actualizar y eliminar, un rifle de francotirador de su interes, puede ser de la familia de los barrett o dragunov
